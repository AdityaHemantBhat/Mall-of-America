# Mall of America | Executive Strategic Vision

An immersive, high-performance digital sales deck designed to showcase the scale, impact, and future-forward vision of Mall of America. This project is engineered to provide an executive-level experience, blending cinematic storytelling with high-density data visualization.

## Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/) (Vanilla CSS performance with modern utilities)
- **Animation**: [GSAP](https://gsap.com/) (ScrollTrigger, Flip) & [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Playfair Display (Executive), Inter (Body), JetBrains Mono (Technical/Data)

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository:**
   ```bash
   git clone <your-repo-link>
   cd <repo-folder>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## Design Decisions

The design follows an **"Executive HUD" (Heads-Up Display)** aesthetic. The goal was to move away from traditional corporate presentations and toward a "Command Center" feel—where data feels alive and property scale feels tangible.

- **Cinematic Pacing**: Used a custom preloader and page transition system to ensure every interaction feels deliberate and premium.
- **Glassmorphism**: Utilized backdrop blurs and subtle borders to create depth, mimicking high-end luxury interfaces.
- **Fluid Motion**: Implemented "liquid" scroll behaviors using GSAP, ensuring that sections transition based on user engagement rather than static clicks.
- **Dark Mode Excellence**: A deep charcoal and gold palette was chosen to evoke the "Mall of America" brand identity while maintaining a sophisticated, high-contrast visual hierarchy.

## AI Implementation

This project was built using a "Human-AI Synergy" approach, leveraging advanced AI tools to push the boundaries of a standard web application:

- **Architectural Visualizations**: All high-fidelity imagery was generated or enhanced using AI to create "hyper-realistic" versions of the property that feel cinematic and aspirational.
- **Motion Orchestration**: AI was utilized to calculate complex GSAP animation timelines and SVG path manipulations (like the page transition curve), ensuring frame-perfect execution.
- **Component Engineering**: The modular architecture (Property Blueprint, Venue Control, Media Network) was developed with AI assistance to ensure clean, performant, and DRY code.
- **Content Strategy**: AI helped refine the data-driven copy to maintain a consistent, executive tone across all 520+ retail touchpoints and stats.

## Key Features

- **Property Blueprint**: An interactive SVG-based layout of the mall’s infrastructure.
- **Retail Landscape**: A horizontal-scrolling experience showcasing luxury anchors and flagship partners.
- **Venue Control**: A technical dashboard interface for event and activation spaces.
- **Media Network**: Dynamic visualization of the mall's digital reach and visitor demographics.
